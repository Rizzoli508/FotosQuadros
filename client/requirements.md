## Packages
react-dropzone | Robust drag-and-drop file upload capabilities for the photo sections
framer-motion | Essential for premium, smooth page transitions and micro-interactions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes safely

## Notes
The photo upload simulates sending files by mapping the file objects to a JSON array for the `photos` field in the database, as an actual file storage endpoint is not provided in the backend schema.
